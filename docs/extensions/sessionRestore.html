API 参考：chrome.sessionRestore

类型

ClosedEntry

( object )

ClosedEntry 的属性

id ( integer )
The ID of the browser session.
timestamp ( integer )
The time when the window or tab was closed, represented in milliseconds since the epoch
tab ( optional tabs.Tab )
The closed tabs.Tab, if this entry describes a tab. Either this or ClosedEntry.window will be set.
window ( optional windows.Window )
The closed windows.Window, if this entry describes a window. Either this or ClosedEntry.tab will be set.

方法

getRecentlyClosed

chrome.sessionRestore.getRecentlyClosed(object options, function callback)

Gets the list of recently closed tabs and/or windows.

参数

options ( optional object )
maxResults ( optional integer )
The maximum number of entries to be fetched in the requested list. Omit to fetch the maximum number of entries (25).
entryType ( optional enumerated string ["tab", "window"] )
The type of entry to fetch. Omit to fetch all types of entries.
callback ( function )

回调函数

callback 参数应该指定一个如下形式的函数： 

function(array of ClosedEntry entries) {...};
entries ( array of ClosedEntry )
The list of closed entries in reverse order that they were closed (the most recently closed tab or window will be at index 0). If entryType is specified the entries will be only of that type, otherwise the entries may contain either tabs or windows.List of ClosedEntry representing the closed tab/window for the session.

restore

chrome.sessionRestore.restore(integer id, function callback)

Reopens a ClosedEntry, with an optional callback to run when the entry has been restored.

参数

id ( optional integer )
The id of the ClosedEntry to restore.
callback ( optional function )

回调函数

如果您指定了 callback 参数，它应该指定一个如下形式的函数： 

function() {...};
