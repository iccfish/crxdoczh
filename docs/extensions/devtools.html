chrome.devtools.* API

目录

  1. 如何使用开发者工具 API
  2. 更多信息
  3. 例子

以下 API 模块为扩展 Chrome 浏览器的开发者工具提供支持：

  • devtools.inspectedWindow
  • devtools.network
  • devtools.panels

如何使用开发者工具 API

  1. 在您的扩展程序的清单文件中指定 "devtools_page" 字段。 
    {
  "name": ...
  "version": "1.0",
  "minimum_chrome_version": "10.0",
  "devtools_page": "devtools.html",
  ...
}
  2. 每打开一个开发者工具窗口，都会为您的扩展程序清单文件中 devtools_page 指定的页面创建一个实例。这一页面可以使用 devtools.panels API，向开发者工具的窗口添加其他扩展程序页面，作为面板和侧边栏。
  3. chrome.devtools.* API 模块仅对于在开发者工具窗口内载入的页面可用，内容脚本和其他扩展程序页面没有这些 API。因此，这些 API 仅在开发者工具窗口的生命周期内可用。
  4. 可用于开发者工具窗口中的扩展程序页面的 API 包括所有以上列出的 chrome.devtools 模块以及 chrome.extension API。其他扩展程序 API 对于开发者工具的页面不可用，但是您可以通过向您的扩展程序的后台页面发送请求来调用它们，与内容脚本中所做的类似。
  5. 还有一些开发者工具 API 仍然是实验性的，有关实验性 API 的完整列表以及如何使用它们的指南请参见 chrome.experimental.* API。
  6. 给我们反馈！您的评论和建议有助于我们改善这些 API。

更多信息

有关扩展程序可以使用的标准 API，请参见 chrome.* API 以及 其他 API。
