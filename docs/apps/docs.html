文档版本

通常，您应该在 https://crxdoc-zh.appspot.com/extensions/<文件名>（例如 https://crxdoc-zh.appspot.com/extensions/overview.html）。然而，如果您需要查看最新的文档，或者正在使用的 Google Chrome 浏览器版本与其他大多数人不同（也许是 Dev 分支），您可能希望使用另一种 URL（例如 .../dev/extensions/overview.html）。下表列出了文档位置，并解释了它们的区别。

URL 版本
.../extensions/... 也许就是您正在使用的版本，该文档描述的是最稳定版本的扩展程序 API。
.../beta/extensions/...

用于 Beta 分支版本的 Google Chrome 浏览器的文档。

注意：Beta 分支中的 API 可能会更改。

.../dev/extensions/...

用于 Dev 分支版本的 Google Chrome 浏览器的文档。该版本可能会包含问题修复和特色内容，与当前文档相关，但是还没有整合进来。

注意：Dev 分支中的 API 可能会更改。

.../trunk/extensions/... 这是最新的文档。如果您使用 Canary 或最新版本的 Chromium，或者对正在开发的新特性很有好奇心，请来这儿看看。该版本也会包含问题修复和特色内容，与当前文档相关，但是还没有整合进来。

注意：trunk 版本的文档还是预备文档，可能有错误。
