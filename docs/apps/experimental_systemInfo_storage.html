chrome.experimental.systemInfo.storage

目录

  1. API 参考：chrome.experimental.systemInfo.storage
    1. 类型
      1. StorageUnitType
      2. StorageUnitInfo
      3. StorageChangeInfo
    2. 方法
      1. get
    3. 事件
      1. onAvailableCapacityChanged

API 参考：chrome.experimental.systemInfo.storage

类型

StorageUnitType

( enumerated string ["unknown", "harddisk", "removable"] )

StorageUnitInfo

( object )

StorageUnitInfo 的属性

id ( string )
The unique id of the storage unit.
type ( StorageUnitType )
The type of storage device. The value is one of the constants defined for this type.
capacity ( double )
The total amount of the storage space, in bytes.
availableCapacity ( double )
The available amount of the storage space, in bytes.

StorageChangeInfo

( object )

StorageChangeInfo 的属性

id ( string )
The uniue id of the storage unit already changed.
availableCapacity ( double )
The new value of the available capacity.

方法

get

chrome.experimental.systemInfo.storage.get(function callback)

Get the storage information from the system. The argument passed to the callback is an array of StorageUnitInfo objects.

参数

callback ( function )

回调函数

callback 参数应该指定一个如下形式的函数： 

function(array of StorageUnitInfo info) {...};
info ( array of StorageUnitInfo )

事件

onAvailableCapacityChanged

chrome.experimental.systemInfo.storage.onAvailableCapacityChanged.addListener(function(StorageChangeInfo info) {...});

Fired when the storage information is updated.

监听器参数

info ( StorageChangeInfo )
The changed information for the specified storage unit
