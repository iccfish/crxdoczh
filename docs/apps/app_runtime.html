chrome.app.runtime

目录

  1. API 参考：chrome.app.runtime
    1. 类型
      1. LaunchItem
      2. LaunchData
    2. 事件
      1. onLaunched
      2. onRestarted
    3. 示例应用
Description: Use the chrome.app.runtime module to manage the app lifecycle. The app runtime manages app installation, controls the event page, and can shut down the app at anytime.
Availability: Google Chrome 23
Permissions: None
Learn more: Manage App Lifecycle
Chrome Office Hours: Introduction to Chrome Apps

API 参考：chrome.app.runtime

类型

LaunchItem

( object )

LaunchItem 的属性

entry ( fileentry )
FileEntry for the file.
type ( string )
The MIME type of the file.

LaunchData

( object )

LaunchData 的属性

id ( optional string )
The id of the file handler that the app is being invoked with.
items ( optional array of LaunchItem )

事件

onLaunched

chrome.app.runtime.onLaunched.addListener(function(LaunchData launchData) {...});

Fired when an app is launched from the launcher.

监听器参数

launchData ( optional LaunchData )

onRestarted

chrome.app.runtime.onRestarted.addListener(function() {...});

Fired at Chrome startup to apps that were running when Chrome last shut down.
