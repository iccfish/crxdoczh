Description: Use the chrome.serial module to read from and write to a device connected to a serial port.
Availability: Google Chrome 23
Permissions: "serial"
Learn more: Accessing Hardware Devices

API 参考：chrome.serial

类型

OpenOptions

( object )

OpenOptions 的属性

bitrate ( integer )
The requested bitrate of the connection to be opened. For compatibility with the widest range of hardware, this number should match one of commonly-available bitrates, such as 110, 300, 1200, 2400, 4800, 9600, 14400, 19200, 38400, 57600, 115200. There is no guarantee, of course, that the device connected to the serial port will support the requested bitrate, even if the port itself supports that bitrate.

OpenInfo

( object )

OpenInfo 的属性

connectionId ( integer )
The id of the opened connection.

ReadInfo

( object )

ReadInfo 的属性

bytesRead ( integer )
The number of bytes received, or a negative number if an error occurred. This number will be smaller than the number of bytes requested in the original read call if the call would need to block to read that number of bytes.
data ( arraybuffer )
The data received.

WriteInfo

( object )

WriteInfo 的属性

bytesWritten ( integer )
The number of bytes written.

ControlSignalOptions

( object )

ControlSignalOptions 的属性

dtr ( optional boolean )
Serial control signals that your machine can send. Missing fields will be set to false.
rts ( optional boolean )
dcd ( optional boolean )
Serial control signals that your machine can receive. If a get operation fails, success will be false, and these fields will be absent.

DCD (Data Carrier Detect) is equivalent to RLSD (Receive Line Signal Detect) on some platforms.
cts ( optional boolean )

方法

getPorts

chrome.serial.getPorts(function callback)

Returns names of valid ports on this machine, each of which is likely to be valid to pass as the port argument to open(). The list is regenerated each time this method is called, as port validity is dynamic.

参数

callback ( function )
Called with the list of ports.

回调函数

callback 参数应该指定一个如下形式的函数： 

function(array of string ports) {...};
ports ( array of string )

open

chrome.serial.open(string port, OpenOptions options, function callback)

Opens a connection to the given serial port.

参数

port ( string )
The name of the serial port to open.
options ( optional OpenOptions )
Connection options.
callback ( function )
Called when the connection has been opened.

回调函数

callback 参数应该指定一个如下形式的函数： 

function(OpenInfo openInfo) {...};
openInfo ( OpenInfo )

close

chrome.serial.close(integer connectionId, function callback)

Closes an open connection.

参数

connectionId ( integer )
The id of the opened connection.
callback ( function )
Called when the connection has been closed.

回调函数

callback 参数应该指定一个如下形式的函数： 

function(boolean result) {...};
result ( boolean )

read

chrome.serial.read(integer connectionId, integer bytesToRead, function callback)

Reads a byte from the given connection.

参数

connectionId ( integer )
The id of the connection.
bytesToRead ( integer )
The number of bytes to read.
callback ( function )
Called when all the requested bytes have been read or when the read blocks.

回调函数

callback 参数应该指定一个如下形式的函数： 

function(ReadInfo readInfo) {...};
readInfo ( ReadInfo )

write

chrome.serial.write(integer connectionId, arraybuffer data, function callback)

Writes a string to the given connection.

参数

connectionId ( integer )
The id of the connection.
data ( arraybuffer )
The string to write.
callback ( function )
Called when the string has been written.

回调函数

callback 参数应该指定一个如下形式的函数： 

function(WriteInfo writeInfo) {...};
writeInfo ( WriteInfo )

flush

chrome.serial.flush(integer connectionId, function callback)

Flushes all bytes in the given connection's input and output buffers.

参数

connectionId ( integer )
The id of the connection.
callback ( function )
Called when the flush is complete.

回调函数

callback 参数应该指定一个如下形式的函数： 

function(boolean result) {...};
result ( boolean )

getControlSignals

chrome.serial.getControlSignals(integer connectionId, function callback)

参数

connectionId ( integer )
callback ( function )
Returns a snapshot of current control signals.

回调函数

callback 参数应该指定一个如下形式的函数： 

function(ControlSignalOptions options) {...};
options ( ControlSignalOptions )

chrome.serial.setControlSignals(integer connectionId, ControlSignalOptions options, function callback)

参数

options ( ControlSignalOptions )
callback ( function )
Returns true if operation was successful.

回调函数

function(boolean result) {...};
