在您开始之前

目录

  1. 从哪里开始

该文档向您介绍了如何编写打包应用。然而所有开发者都应该知道用于打包应用的新 API 正在以开发者预览的方式发布，这意味着它们每天都在变化，您现在阅读到的任何内容在不远的将来都有可能变化。请随时保持阅读最新的 API 参考和文档。如果您遇到任何问题，欢迎在 #chromium-apps（英文）反馈。

注意：如果您以前曾经写过打包应用，您遗留的打包应用仍然会像以前那样继续工作，但是它们无法访问新的 API。

从哪里开始

入门指南是一个很好的起点，它读起来很快，读完所有这三个文档不会超过 10 分钟。入门指南之后，确定哪些与您最相关。基础知识指南涉及了应用与数据生命周期的详情，您也可以阅读 MVC 架构了解有关良好的应用设计的更多内容。我们在代码库中也提供了大量的示例代码，文档中都有直接的链接。

如果您熟悉 Chrome 扩展程序文档，那么参考文档看上去一定很熟悉。打包应用和扩展程序公用同一个平台，它们可以访问许多相同的 API，它们具有相同的清单文件与权限格式。许多参考文档都是共享的，我们已经过滤掉了不能共享的文档的访问。
