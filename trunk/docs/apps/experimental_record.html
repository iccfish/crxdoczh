      1. SessionDetails
      2. ReplayURLsResult
    2. 方法
      1. captureURLs

API 参考：chrome.experimental.record

类型

SessionDetails

( object )

SessionDetails 的属性

extensionPath ( optional string )
Absolute path to an unpacked extension to run in the subbrowser session.

ReplayURLsResult

( object )
Return value for Replay callback

ReplayURLsResult 的属性

runTime ( double )
Time in milliseconds to complete all runs.
stats ( string )
Full multiline dump of output stats, showing one statistic per line, comprising an abbreviated statistic name and its value (e.g. vmsize_f_b= 696164352 bytes for final vm size). This is ugly, and will be changed shortly.
errors ( array of string )
List of errors during replay. Presently, this should only be abnormal browser termination for unexpected reasons.

方法

captureURLs

chrome.experimental.record.captureURLs(string captureName, array of string urls, function callback)

参数

captureName ( string )
Unique name of the capture.
urls ( array of string )
URL list to visit during capture.
callback ( optional function )
Called when capture has completed.

回调函数

如果您指定了 callback 参数，它应该指定一个如下形式的函数： 

function(array of string errors) {...};
errors ( array of string )
List of any URLs that failed to load, one error per textline, along with failure reason (e.g. unknown domain). Also may include general abnormal-exit message if the subbrowser run failed for other reasons.

replayURLs

chrome.experimental.record.replayURLs(string captureName, integer repeatCount, SessionDetails details, function callback)

参数

captureName ( string )
Unique name of capture. Use to determine cache.
repeatCount ( integer )
details ( optional SessionDetails )
callback ( optional function )
Called when playback has completed.

回调函数

如果您指定了 callback 参数，它应该指定一个如下形式的函数： 

function(ReplayURLsResult result) {...};
result ( ReplayURLsResult )
